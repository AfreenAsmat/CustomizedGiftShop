# Customized Gift Shop Management System

A full-featured ASP.NET Core MVC web application for managing a customized gift shop.
The system supports product management, customer handling, order processing, POS billing, inventory tracking, authentication, role-based authorization, and sales reporting.


## ✨ Features

### 🔐 Authentication & Authorization

* User Registration & Login
* Role-based access control
* Admin and Staff roles
* Secure authentication using ASP.NET Identity


### 📦 Product Management

* Add new products
* Edit product details
* Delete products
* Upload product images
* Inventory/stock management
* Out-of-stock handling


### 👥 Customer Management

* Add customers
* Edit customer information
* Delete customers
* View customer list


### 🧾 Order Management

* Create customer orders
* Delivery & Takeaway order types
* Edit pending orders
* Cancel or complete orders
* Automatic stock updates
* Invoice generation


### 💳 POS (Point of Sale)

* Interactive billing interface
* Add products to cart
* Real-time total calculation
* Walk-in customer support
* Checkout using AJAX & JSON
* Automatic invoice redirection


### 📊 Sales Reporting

* Total revenue calculation
* Total orders tracking
* Total items sold
* Completed orders report
